Links

Em CSS as classes que modificam os links são:

a
serve para todo o texto da página que utilize a tag a, mesmo que não seje um link.
a:link
afeta os links antes de clicados.
a:hover
afeta o link no momento em que o mouse passa sobre ele. Se você usar a classe a:visited a classe a:hover não terá efeito sobre links já visitados.
a:active
afeta o link no momento em que é clicado.
a:visited
afeta os links que já foram visitados.
