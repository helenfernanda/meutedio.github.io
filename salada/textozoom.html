Você pode modificar as primeiras linhas do código:

text=new Array ('','','','','')
Dentro dos parênteses, entre aspas simples e separados por vírgula escreva os textos que aparecerão.
var numText=5
Aqui você especifica o número de textos que você escreveu acima (cada par de aspas é um texto).
color=new Array ('','','','','','','','','','')
Dentro das aspas você escreve as cores do último texto. A última cor é a cor definitiva.
var numColors=10
É o número de cores que você escreveu acima.
var endSize=50
É o tamanho do texto.
var Zspeed=75
É a velocidade em que o texto se aproxima.
var Cspeed=145
É a velocidade em que as cores do último texto mudam.
var font='garamond, times new roman'
É a fonte do texto.

Todo o texto que deve vir abaixo do texto em zoom deve vir abaixo da tag div style='margin-top:100;'. Se o número especificado no parâmetro margin-top for menor que o espaço que o texto em zoom ocupa ou simplesmente não for especificado, o texto em zoom ficará por cima do que estiver embaixo. Isso pode ser legal se você quiser que o texto em zoom fique sobre algum outro texto ou sobre uma figura.
