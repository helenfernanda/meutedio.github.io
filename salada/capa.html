Sobre o site

Net Mania era uma seção dentro do meu site pessoal. O conteúdo dessa seção cresceu tanto que acabou se tornando um novo site que agora se chama Salada Digital. Espero que gostem!

Não prometo atualizações constantes. O meu objetivo é partilhar um pouco do que sei, então o site será atualizado a medida que eu aprender e/ou criar coisas novas e tiver tempo de colocar no site.

Tudo o que está neste site foi testado no Internet Explorer 6, é possível que algo não funcione se você usa Internet Explorer mais antigo ou outro navegador.

Para "pegar" os códigos deste site utilize os atalhos do teclado (para Windows):

  • Ctrl + F para localizar algo na página ou no código
  • Ctrl + A seleciona todo o texto se o cursor já estiver dentro da caixa
  • Ctrl + C copia o texto selecionado
  • Ctrl + X recorta o texto selecionado
  • Ctrl + V cola o texto copiado ou recortado no local desejado
  • Ctrl + Z desfaz o último comando

Para testar os exemplos do site use o Notepad (Bloco de notas) do Windows ou qüalquer programa similar. Cole o exemplo na janela do bloco de notas e salve com a extensão .html. Depois abra esse mesmo exemplo em seu navegador (de preferência o Internet Explorer).
