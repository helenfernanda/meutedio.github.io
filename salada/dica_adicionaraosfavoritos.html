Adicionar aos favoritos

Este código gera um link para que o visitante possa facilmente adicionar teu blog/site aos Favoritos de seu browser. Modifique:

  • onde está http://www.helenfernanda.kit.net escreva o endereço correto de seu blog/site.
  • onde está escrito Helen Fernanda escreva o nome correto de seu blog/site.
  • onde está escrito Adicione aos favoritos escreva o texto desejado.
Adicione aos favoritos

Este código gera um botão com a mesma utilidade. Modifique:

  • onde está escrito http://www.helenfernanda.kit.net digite o endereço de seu blog/site.
  • onde está escrito Helen Fernanda digite o nome de seu blog/site.
  • outras modificações são opcionais como cores, bordas, fonte e mensagem.
