Grupos de discussão

Um grupo de discussão é uma página e um endereço de e-mail que permitem a um grupo de pessoas interagir através de enquetes, arquivos, fotos e principalmente pela troca de mensagens. Todos os serviços listados estão disponíveis em português.

  • Yahoo! Grupos . br.groups.yahoo.com . saiba mais
    • PÁGINA E ENDEREÇOS DO GRUPO - No Yahoo! Grupos tudo é muito objetivo e intuitivo. O nome que você escolhe para o grupo não é necessariamente o endereço do mesmo. Confira esse exemplo:
      • Nome escolhido: Salada digital
      • Endereço escolhido: saladadigital
      • Página inicial: br.groups.yahoo.com/group/saladadigital
      • Página para cadastro/descadastro: br.groups.yahoo.com/group/saladadigital/join
      • Página de membros: br.groups.yahoo.com/group/saladadigital/members
      • Endereço para cadastro: saladadigital-subscribe@yahoogrupos.com.br
      • Endereço do grupo: saladadigital@yahoogrupos.com.br
      • Endereço para descadastro: saladadigital-unsubscribe@yahoogrupos.com.br

      No exemplo acima citado o nome do grupo é igual ao endereço escolhido, mas isso não regra. Assim, se meu grupo se chama "Mulheres encalhadas há anos", posso escolher "mulheresencalhadas", "homensindecisos" ou "meha" como endereço (desde que esses endereços já não tenham sido escolhidos por outra mente tão brilhante como a minha). Assim podem existir vários grupos com o mesmo nome, mas com endereços diferentes.

    • RESTRIÇÕES - Qualquer endereço eletrônico pode participar do grupo enviando e recebendo mensagens, mas somente uma id do Yahoo! tem acesso aos recursos da página do grupo. LEGAL: Você pode participar do grupo com um e-mail qualquer para receber e enviar mensagens, mas utilizar sua id do Yahoo! para acessar os recursos do grupo, já que seu cadastro no Yahoo! permite vários e-mails adicionais. Se você participa de vários grupos pode utilizar um endereço diferente para cada um deles, todos vinculados a uma único cadastro do Yahoo!. Também é possível ter uma id diferente para cada grupo, já que o limite do Yahoo! é de sete ids por cadastro.
    • RECURSOS DA PÁGINA - Páginas de: arquivos, fotos, links, enquetes, banco de dados, membros. É possível configurar as cores do grupo com um dos temas prontos ou personalizar com as cores que bem entender. Também é possível escolher uma imagem para aparecer na página inicial, junto à descrição do grupo.
    • RECURSOS DE MENSAGENS - Na página de gerenciamento é possível configurar o grupo para que apenas o moderadores enviem, para que todos os membros enviem, para que qualquer um envie mensagens, para que todas as mensagens sejam moderadas antes de distribuídas. Também existe a opção de que os arquivos enviados ao grupo sejam removidos (quando essa opção é ativada, toda a formatação HTML também é removida das mensagens).

  • MSN Grupos . groups.msn.com . saiba mais
    • PÁGINA E ENDEREÇOS DO GRUPO - Se você escolhe um nome com caracteres especiais (acento, cedilha, crase), o endereço do grupo vira um monte de códigos que ninguém sabe de onde vem. Caso você simplifique o serviço e coloque um nome sem caracteres especiais, vale a regra exemplificada abaixo:
      • Nome escolhido: Correio Mania
      • Endereço do grupo: correiomania (não foi escolhido, mas sim gerado pelo sistema)
      • Página inicial: groups.msn.com/correiomania
      • Endereço do grupo: correiomania@groups.msn.com
    • RESTRIÇÕES - Só é possível cadastrar-se no grupo tendo um Net Passport. Os endereços@hotmail.com e os endereços@msn.com são Net Passport desde o nascimento, mas os outros precisam ser convertidos, quer dizer, cadastrados na página passport.net. Depois do cadastro (que só é efetuado quando o endereço é verificado) basta entrar na página do grupo e clicar em "Ingressar Já". Alguém também pode entrar em um grupo se for convidado.
    • RECURSOS DA PÁGINA - Em um grupo MSN você pode criar várias páginas, listas, bancos de dados, álbuns de fotos, etc. Se souber HTML e CSS então... A imaginação correrá solta.
    • RECURSOS DAS MENSAGENS - A troca de mensagens, que é o objetivo de um grupo, é a parte mais chata no MSN. Chata porque as mensagens enviadas/recebidas ganham uma moldura horrorosa que as deixam mais largas e mais pesadas. Muito tosco aquilo!
