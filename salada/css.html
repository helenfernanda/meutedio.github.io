CSS

A CSS, sigla de Cascading Style Sheets - Folhas de Estilo em Cascata -, é uma lingüagem que foi criada para complementar o uso de HTML na criação de sites. Na CSS não há tags, mas sim propriedades.

Para inserir uma folha de estilos em um documento HTML deve-se usar as tags <style> e </style> dentro do cabeçalho da página.

Para testar o exemplo abaixo, copie o código, cole no bloco de notas e salve como qualquernome.html. 

<html>
 <head>
  <title>Primeira página</title>
 <style>
   body {background-color:#cc0000;
         color:white;
         font-family:lucida sans;
         font-size:15px;}
   h1 {background:#0000cc;}
   a {color:pink;}
   a:link {color:yellow;}
   a:hover {color:white;}
   a:active {color:lime;}
  </style>
 </head>
 <body>
 <h1>Testando CSS</h1>
 <a href="http://www.sitenetmania.cjb.net"
 target='_blank'>sitenetmania.cjb.net</a>
 </body>
</html>

Para determinar o estilo dentro de uma tag utiliza-se o parâmetro style:

<p style="color:teal; font-family:verdana;">CSS na tag</p>
