Posição

Utilizando margens, alinhamento, indentação e outras propriedades é possível definir a posição relativa de textos, tabelas, listas, imagens e outros elementos de uma página.

Com CSS é possível ainda definir a posição absoluta de qualquer elemento dentro da página utilizando a propriedade position com valor igual a absolute. Relative é o valor padrão.

Left, right, top, bottom, width e height são algumas das propriedades que definem a posição absoluta do elemento e os valores podem ser dados em pixels, polegadas, porcentagem, etc.
