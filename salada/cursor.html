Cursores

Coloque o cursor dentro do retângulo para ver o modelo.

Para utilizar o cursor você tem duas opções:

  1. Clicar no link (que está dentro do retângulo) para salvar o cursor desejado e depois salvá-lo no diretório de seu site. Esta opção não funciona em blogs gratuitos porque poucos deles oferecem a opção de envio de arquivos e os que permitem não aceitam o envio de cursores.
  2. Copiar o código abaixo substituindo 00.cur pelo nome do cursor (que está dentro do retângulo). A classe body faz com que o cursor apareça em toda a extensão da página, mas você pode substituí-la por p (parágrafo), table (tabela), a (link), etc.
