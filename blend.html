Esta página é um exemplo do efeito de transição Blend. Para utilizar este efeito no seu site, coloque o código abaixo entre as tags <head> e </head>:

<meta http-equiv="Page-Enter" content="blendTrans(Duration=1.0)">
<meta http-equiv="Page-Exit" content="blendTrans(Duration=1.0)">

Para mais códigos, visite este site.
