Esta metatag é para dizer quem é(são) o(s) autor(es) do site. Onde está escrito "Helen Fernanda" escreva teu(vossos) nome(s):

<meta name="Author" content="Helen Fernanda">

Esta indica qual programa você utilizou para criar o site. Onde está escrito "Notepad" escreva o nome do programa que você usa.

<meta name="Generator" content="Notepad">
