Barra de rolagem (scrollbar) colorida

Esta página é um anexo do post CSS: barra de rolagem colorida no Google Chrome.

Aqui temos um exemplo de barra de rolagem (scrollbar) colorida que deve funcionar no Google Chrome e em outros navegadores que usam webkit, como Opera e Flock.

Código para a folha de estilos:

::-webkit-scrollbar {background: #ffffff;} /* barra de rolagem toda */
::-webkit-scrollbar-thumb {background: #000000;} /* parte da barra que se movimenta */
::-webkit-scrollbar-button {background:#99cc00;} /* ponta da barra de rolagem */
::-webkit-scrollbar-track {background:#ffffff;} /* fundo da barra de rolagem, parte fixa */
::-webkit-scrollbar-track-piece {background:#ffffff;} /* fundo da barra de rolagem, parte que ainda falta rolar */
::-webkit-scrollbar-corner {background:#99cc00;} /* canto inferior direito da barra de rolagem */

Confira também:

  • Barra de rolagem com borda
  • Barra de rolagem com imagem
  • Barra de rolagem com sombra
  • Barra de rolagem que muda de cor
  • Barra de rolagem listrada que muda de cor
