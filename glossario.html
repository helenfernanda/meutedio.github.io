Glossário

  • blink: cintilação
  • browser: navegador
  • desktop: área de trabalho
  • doll: boneca
  • e-mail [i-meil]: correio eletrônico
  • FTP (File Transfer Protocol): Protocolo de Transferência de Arquivos
  • GIF (Graphics Interchange Format): Formato de Gráficos de Permuta
  • HTML (HyperText Markup Language): Linguagem de hipertexto
  • HTTP (HyperText Transfer Protocol): Protocolo de Transferência de Hipertexto
  • homepage [roumpeigi]: página de origem, página pessoal, primeira página
  • JPEG (Joint Photographic Experts Group): Grupo Especializado Fotográfico Articulado
  • link: elo, ligação, vínculo
  • messenger: mensageiro
  • MIME (Multipurpose Internet Mail Extensions): Extensões para correio eletrônico com múltiplos usos
  • newsgroups: grupos de notícias, grupos de discussão
  • PERL (Pratical Extraction and Reporting Language): Extração Prática e Linguagem Descritiva
  • paint [peint]: pintura, pintar, colorir
  • screensaver [scrinseiver]: proteção de tela
  • script: texto de um programa
  • SGML (Standard General Markup Language): Linguagem padrão geral (de hipertexto)
  • site [sait]: posição, lugar, terreno, espaço, sítio
  • URL (Uniform Resource Locator): Localidade uniforme de recursos
  • VRML (Virtual Reality Modeling Language): Linguagem Modelada de Realidade Virtual
  • wallpaper [uolpeiper]: papel de parede
  • WAIS (Wide Area Information Server): Ampla Área de Informação ao Usuário
  • WWW (World Wide Web): Ampla Rede Mundial
