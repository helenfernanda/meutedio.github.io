Links

Para criar um link deve ser utilizada a tag <a> que tem os seguintes parâmetros:

  • href define para onde o link leva ao ser clicado. Se href=# o link abre a mesma página onde se encontra o link, na mesma ou em outra janela.
  • title é opcional e serve para escrever um texto que aparecerá qüando o cursor estiver sobre o link.
  • name serve para você nomear um link.
  • target define em que janela a página será aberta, caso o link chame outra página. Se a página deve abrir na janela atual, basta omitir este parâmetro ou usar _self. Caso você queira que a página abra em uma nova janela você pode dar qüalquer nome a esta nova janela ou utilizar _blank. Caso o site utilize frames ou iframes _parent abre a página no próximo frame e _top abre na janela principal.

Links para e-mail só funcionam se o visitante tiver um programa de e-mail instalado e configurado em sua máquina.

Exemplo Código
Esta página <a href=# target=jornalismo title='Esta página em outra janela'>Esta página</a>
Página Inicial <a href=capa.html target=luiz title='Página Inicial'>Página Inicial</a>
Helen Fernanda
 <a href=http://www.helenfernanda.cjb.net target=lua title='Site da Helen Fernanda'>Helen Fernanda</a>
helen@pur.com.br <a href="mailto:helen@pur.com.br?subject=Net Mania" title='E-mail para Helen Fernanda'>helen@pur.com.br</a>

Você pode criar um link para uma seção dentro da própria página:
Isto você coloca no link: Isto você coloca no local para onde o link deve levar:
<a href=#nome>Clique aqui para ir lá.</a> <a name=nome></a>Aqui vem o que você quiser.

Com HTML as cores dos links podem ser definidas dentro da tag <body> com os seguintes parâmetros:

  • link é a cor dos links que ainda não foram visitads
  • alink é a cor do link no momento em que é clicado
  • vlink é a cor do link já visitado

<body bgcolor=black link='deeppink' alink='#ffaaff' vlink='rgb(255 40 255)'>
