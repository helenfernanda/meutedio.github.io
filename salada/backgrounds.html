Como utilizar os backgrounds

  1. Clique nos números do índice para abrir as páginas.
  2. Passe o cursor do mouse sobre a textura para vê-la no background.
  3. Clique sobre a imagem com o botão direito para salvá-la no local desejado.
  4. Para inserir uma imagem no background de sua página utilize o parâmetro background dentro da tag <body>, como no exemplo:

    <body background="imagem.gif">

    Ou utilize a propriedade background-image dentro do parâmetro style:

    <body style="background-image:'imagem.gif';">

    Sendo que imagem.gif é o endereço correto da figura, incluindo a extensão.
