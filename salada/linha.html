Linha

Para organizar o conteúdo de seu site as linhas são muito necessárias. A tag que quebra o texto e gera linhas transparentes é <br> (break).

Exemplo Código
Um gera quebra de texto.
Dois gera uma linha transparente:

Três gera duas linhas transparentes:


E assim sucessivamente...
<center>Um gera quebra de texto.<br>Dois gera uma linha transparente:<br><br>Três gera duas linhas transparentes:<br><br><br>E assim sucessivamente...</center>

Para gerar uma linha visível a tag é <hr> (horizontal rule). Para ela combinar com o template de seu site você pode acrescentar dentro da tag as seguintes propriedades:

  • noshade, para que se torne uma linha sólida (só é necessária se não for especificada uma cor)
  • color, para definir a cor
  • size, para definir a largura da linha (o padrão é 2)
  • width, para definir a extensão da linha (o padrão é 100%)
  • align, para definir o alinhamento (o padrão é center)
<hr color=skyblue size=1 width=80% align=center>


<hr color=rgb(255 0 255) size=5 width=50% align=left>


<hr color=#ff6600 size=10 width=300 align=right>
